# 🎨 Image Cartoonifier using OpenCV

## 📌 Project Description

This project converts a normal image into a cartoon-style image using Computer Vision techniques like edge detection and color smoothing.

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy

## 🚀 Features

* Convert images into cartoon style
* Edge detection using adaptive threshold
* Smooth color filtering

## 📂 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Image-Cartoonifier.git
```

2. Navigate to folder

```bash
cd Image-Cartoonifier
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the project

```bash
python cartoonifier.py
```
